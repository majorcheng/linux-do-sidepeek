# 更新日志

这个文件只记录会影响实际使用体验的变化。

## [Unreleased] - 2026-03-10

### 新增
- 支持抽屉内图片点击全屏放大预览；点击遮罩层、关闭按钮或按 `Esc` 可退出，不再默认跳到独立图片页。

### 修复
- 修复复用同一个抽屉切换到其他主题时，旧图片预览遮罩可能残留在新内容上方的问题。
- 修复带普通外链的图片在抽屉里被误拦截的问题；现在只有图片本身或其原图链接会触发抽屉内预览，普通链接仍按原目标打开。

### 致谢
- 感谢 linux.do 群友 `zhoudashuai` 提出“侧边栏图片点击后改成抽屉内全屏放大、点击遮罩层退出”的优化建议，并推动了这次体验补齐与相关修复。

## [0.2.0] - 2026-03-08

### 修复
- 修复“侧边预览不计浏览”的问题；现在智能预览和整页模式都会尽量按站点正常进帖的方式触发论坛计数。
- 修复切换智能预览 / 整页模式时可能重复计浏览的问题；现在同一次抽屉会话里，同一主题不会因为切模式被重复计数。
- 修复部分页面“右侧滑窗不弹出”的问题；现在首页、搜索页、分类页、标签页、用户发帖流等更多带主题链接的页面都可以直接侧边预览。
- 修复主题详情页里的推荐主题等标题链接不能侧边预览的问题；现在只排除正文、编辑器和弹层里的链接，不会误伤正常主题标题。
- 修复某些场景下误触发右侧滑窗的问题；帖子正文、编辑器、弹层菜单等区域里的主题链接默认不会被接管。
- 修复未读帖、搜索命中等链接总是从首帖打开的问题；现在会尽量保留你点击时对应的楼层位置。
- 修复深链接在“仅首帖”或“首帖 + 最新回复”模式下内容错位的问题；现在不会再把某条回复误当成首帖固定在顶部。
- 修复 `/last`、未读定位等 helper 链接在智能预览里丢失目标位置的问题；现在会尽量保留你点击时的最后位置或目标楼层。
- 修复部分主题链接识别错乱的问题；短格式楼层链接、搜索结果、用户动态里的同主题多条记录，现在不会再跳错帖或让“上一帖/下一帖”定位错乱。

### 体验调整
- 调整回复排序的产品定义；智能预览默认仍是正常顺序，“首帖 + 最新回复”现在明确表示优先展示最新一批回复，不再暗示整帖完整倒序。
- 调整重复点击同一帖子的处理方式；扩展不再自己做长时间去重，是否计入浏览交给论坛本身判断。
- 调整右侧滑窗的适用范围说明；现在强调的是“支持站内可预览的主题链接”，不再只写传统列表页。

## [0.1.0] - 2026-03-07

### 首次发布
- 支持在 Linux.do 右侧滑窗预览帖子，不用频繁进出帖子页。
- 支持智能预览与整页模式切换。
- 支持上一帖、下一帖和快捷键切换。
- 支持预览选项、抽屉宽度调整与设置保存。
